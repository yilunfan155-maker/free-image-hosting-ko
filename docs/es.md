# Alojamiento gratuito de imágenes y generador de URL

Sube una imagen y obtén una URL directa, además de códigos Markdown, HTML y BBCode para documentación, foros, tickets de soporte y archivos README.

- Tool: https://mini-tools.uk/upload
- Output: Direct URL, Markdown, HTML, BBCode
