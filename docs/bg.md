# Безплатен хостинг на изображения и генератор на URL

Качете изображение и получете директен URL, както и Markdown, HTML и BBCode фрагменти за документация, форуми, тикети за поддръжка и README файлове.

- Tool: https://mini-tools.uk/upload
- Output: Direct URL, Markdown, HTML, BBCode

## Note

Не качвайте лични, чувствителни или поверителни изображения в публична услуга за хостинг на изображения.
