# Hospedagem gratuita de imagens e gerador de URL

Envie uma imagem e obtenha uma URL direta, além de códigos Markdown, HTML e BBCode para documentação, fóruns, tickets de suporte e arquivos README.

- Tool: https://mini-tools.uk/upload
- Output: Direct URL, Markdown, HTML, BBCode

## Related searches

## Note

Não envie imagens privadas, sensíveis ou confidenciais para um serviço público de hospedagem de imagens.
