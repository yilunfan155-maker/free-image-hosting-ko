# Hébergement gratuit d’images et générateur d’URL

Importez une image et obtenez une URL directe ainsi que des extraits Markdown, HTML et BBCode pour la documentation, les forums, les tickets de support et les fichiers README.

- Tool: https://mini-tools.uk/upload
- Output: Direct URL, Markdown, HTML, BBCode

## Note

N’importez pas d’images privées, sensibles ou confidentielles vers un service public d’hébergement d’images.
